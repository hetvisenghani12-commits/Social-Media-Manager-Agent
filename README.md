# Social Media Manager Agent

An autonomous social media management system that analyzes past content
performance, compresses audience insights, and schedules future posts.

## Features
- Engagement analysis
- Best posting time detection
- Content type recommendation
- Automatic post scheduling

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. View scheduled_posts.csv output
